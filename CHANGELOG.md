## [1.0.0] — ✨ Polished & Complete

**The full experience.** Flowo levels up from "functional" to "delightful" — a distinct visual identity and mobile-first polish tie everything together.

### Added
- 📱 **Mobile-Responsive Design** — every layout, nav element, and interaction specifically tuned for smaller screens, not just scaled down from desktop.
- 🎨 **Custom Visual Identity** — signal red (`#DB4035`) accent color, Inter typography, and snappy 150ms micro-transitions throughout.
- 💅 Refined Todoist-clarity + Linear.app-precision design language applied app-wide.

### Improved
- ⚡ Snappier perceived performance thanks to consistent, fast transitions.
- 🧭 Overall navigation polish across desktop and mobile.

> 🏁 **Where Flowo stands today** — this is the current shipped version behind the [live app](https://troubler-s-fix.vercel.app/login).

---

## [0.8.0] — 🧠 Smart & Connected

**Flowo gets a brain and a voice.** This release focused on reducing friction and keeping people informed.

### Added
- 🤖 **AI-Assisted Task Creation** — Groq-powered (free-tier) inference turns rough notes into structured tasks in seconds.
- 🔔 **Notifications** — real-time alerts for task assignments, updates, and project activity.
- 🔌 **Extensible Integrations** — groundwork laid for connecting third-party tools into your workflow.

### Improved
- 🧹 Reduced friction between "idea" and "organized task."
- 📡 Better visibility into what's happening across shared projects.

---

## [0.5.0] — 🤝 Team-Ready

**No longer just a solo tool.** Flowo opens up to teams and starts syncing in real time.

### Added
- 👥 **Team Collaboration** — shared projects, task assignments, and team visibility.
- ⚡ **Real-Time Sync** — powered by Supabase Realtime; changes appear instantly for everyone, no manual refresh.

### Improved
- 🔒 Reliability of shared boards — what you see is always current, never a stale snapshot.

---

## [0.1.0] — 🌱 Foundation

**Where it all began.** The first working version of Flowo — simple, focused, and functional.

### Added
- 📋 **Task & Project Management** — create, organize, and track tasks with due dates, priorities, status, and grouping.
- 🗂 Projects as containers to keep related work together.
- 🛠 Core stack established: React + Vite + TypeScript, Tailwind CSS, Supabase, deployed on Vercel.
