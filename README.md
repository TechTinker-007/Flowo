## ✨ Why Flowo

Flowo helps you organize tasks and projects with the simplicity of **Todoist** 📝 and the visual precision of **Linear.app** 📐. It's built on a modern, lightweight stack and designed to stay fast and dependable as it grows — every feature is added deliberately, with existing functionality protected against regressions. 🛡️

## 🚀 Feature Highlights

| | Feature | Description |
|---|---|---|
| 📋 | **Task & Project Management** | Create, organize, and track tasks with due dates, priorities, status, and grouping. |
| 👥 | **Team Collaboration** | Shared projects, task assignments, and visibility across your team. |
| ⚡ | **Real-Time Sync** | Powered by Supabase Realtime — changes appear instantly, everywhere, no refresh needed. |
| 🤖 | **AI-Assisted Task Creation** | Groq-powered (free-tier) inference turns rough thoughts into structured tasks. |
| 🔔 | **Smart Notifications** | Stay informed on task and project activity so nothing slips through. |
| 🔌 | **Extensible Integrations** | Built with room to connect to the other tools in your workflow. |
| 📱 | **Mobile-Responsive** | A fully tuned mobile experience — not just a scaled-down desktop view. |
| 🎨 | **Custom Visual Identity** | Signal red accents, Inter typography, and 150ms micro-transitions. |

📖 For the full breakdown of each feature, see [`FEATURES_DESCRIPTION.md`](./FEATURES_DESCRIPTION.md).
📋 For the quick-glance checklist, see [`FEATURES.md`](./FEATURES.md).

## 🛠 Tech Stack

| | Layer | Technology |
|---|---|---|
| ⚛️ | **Frontend** | React + Vite + TypeScript |
| 💨 | **Styling** | Tailwind CSS |
| 🗄️ | **Backend / Database** | Supabase (Postgres + Realtime + Auth) |
| 🧠 | **AI** | Groq (free-tier inference) |
| ▲ | **Hosting** | Vercel |

## 🎨 Design Language

- 🔴 **Accent color:** Signal red `#DB4035`
- 🔤 **Typography:** Inter
- 🎬 **Motion:** Subtle shadows, fast 150ms transitions for a responsive, tactile feel
- 💡 **Inspiration:** Todoist's clarity + Linear.app's structural precision

## 🗺 Roadmap / Not Yet Included

Some things are intentionally out of scope for now, to keep the product lean while the core experience is refined:

- 💳 Billing / payments
- 👀 Live cursors or multiplayer presence indicators
- 📊 Table or Gantt chart views
- 🗂 Project templates
- 🔐 Single sign-on (SSO)
- 📴 Offline mode / PWA support

These are scoping decisions, not oversights — some may be revisited in future phases.

## 📈 Status

Flowo is actively developed in scoped, additive batches — new features ship without breaking what already works. See [`CHANGELOG.md`](./CHANGELOG.md) for a running history of what's shipped. The live deployment link above always reflects the current version.
